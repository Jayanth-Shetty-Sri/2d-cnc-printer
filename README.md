# 2D CNC Printer

This project demonstrates the design and development of a 2D CNC-based drawing machine using Arduino and stepper motors.

## Demo Video
[Watch Project Demo](https://youtu.be/h5Q0URC9-ik?si=AzcHPxKjdjaX105d)

## Features
- X-Y axis motion using stepper motors  
- Arduino-based control with CNC shield  
- G-code based motion control using GRBL  
- Servo motor for pen lift mechanism

## Tools & Components
- Arduino UNO  
- CNC Shield  
- NEMA 17 Stepper Motors  
- Servo Motor  
- Linear rods and belt mechanism  

## Outcome
The machine can draw images by converting them into G-code and executing precise motion.
